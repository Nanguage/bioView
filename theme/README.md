# Themes

## Simple (default)

Use the following command switch to this theme.

```bash
$ mkdir -p ~/.config/bioview/ ; curl -LJ https://raw.githubusercontent.com/Nanguage/bioView/master/theme/simple.json -o ~/.config/bioview/config.json
```
